# Premium Pet Boarding — Website Scaffold

This is a Vite + React starter scaffold for a pet care and dog boarding website focused on SEO and conversions.

Quick start:

```bash
npm install
npm run dev
```

What this includes:
- Hero section with fast-loading image (uses an external optimized image URL)
- Trust & testimonials section
- Services, Why Choose Us, Pricing, FAQ, Blog list
- Booking/contact form (mailto fallback)
- SEO-focused meta tags and semantic HTML
- Responsive CSS and color palette inspired by pets
